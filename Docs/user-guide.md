# 用户指南

本文档详细介绍如何使用 Shanyu Knowledge 的各项功能。

## 概述

Shanyu Knowledge 是一个用于管理产品文档的静态网站生成工具，基于 Jekyll 构建。

## 核心功能

### 1. 文档管理
- 创建和编辑 Markdown 格式的文档
- 支持文档分类和标签
- 自动生成文档目录

### 2. 搜索功能
- 全文搜索支持
- 关键词高亮显示
- 搜索结果排序

### 3. 版本管理
- 基于 Git 的版本控制
- 文档历史记录
- 版本对比功能

## 使用流程

1. **创建文档**：在 `_docs` 目录下创建 Markdown 文件
2. **编辑内容**：使用 Markdown 语法编写文档
3. **预览效果**：运行 `bundle exec jekyll serve` 预览
4. **发布上线**：推送到 GitHub Pages

## 最佳实践

- 使用清晰的文档结构
- 遵循 Markdown 最佳实践
- 定期更新文档内容
