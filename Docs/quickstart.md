# 快速入门指南

欢迎使用 Shanyu Knowledge！本指南将帮助您快速了解如何使用我们的产品。

## 第一步：安装

请参考 [安装说明](/docs/installation/) 获取详细的安装步骤。

## 第二步：配置

安装完成后，您需要进行基本配置：

1. 打开配置文件 `config.yaml`
2. 设置必要的参数
3. 保存配置

## 第三步：启动

运行以下命令启动服务：

```bash
shanyu start
```
