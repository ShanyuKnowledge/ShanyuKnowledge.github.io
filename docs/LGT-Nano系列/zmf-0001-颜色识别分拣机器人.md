# 颜色识别分拣机器人

## 产品概述

颜色识别分拣机器人是一款基于机器视觉技术的智能分拣设备，能够自动识别物体颜色并进行分类分拣。

## 技术参数

| 参数 | 规格 |
|------|------|
| 识别速度 | 30件/分钟 |
| 识别精度 | 99.5% |
| 支持颜色 | 红、黄、蓝、绿、黑、白等 |
| 工作电压 | 220V AC |

## 功能特性

- 实时颜色识别
- 自动分拣分类
- 支持多种颜色识别
- 支持自定义分拣规则
- 数据统计与分析

## 使用说明

### 启动设备

1. 检查电源连接
2. 打开设备开关
3. 等待系统初始化

### 操作步骤

1. 将待分拣物品放置到传送带上
2. 系统自动识别颜色
3. 机器人进行分拣操作
4. 物品被分类到对应区域
